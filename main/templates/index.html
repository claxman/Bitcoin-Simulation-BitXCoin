BitX-Coin Client Setup

I've designed a simple, easy to understand simulation of Bitcoin network. Here's how it works.



01

Clients/ Users

This simulation has two clients (alice and bob for instance). Clients can generate their wallets and save public/ private keys for future transactions.

02

Make Transaction

Clients can send & receive BitX Coins to and from respectively. It includes genreating wallets and showing transcationsdetails.

03

Transactions

You can access previous transcations using this feature.





BitX-Coin Node Setup

1Transactions

Once the transaction is done, it'll be visible here. Upon mining this block, it'll be added in the blockchain.

2Blockchain

Blockchain ledger will be visible here. It'll contain the transactions and the reward gained by the user upon mining the block.

3Configurations

You can configure diffeerent nodes here so that they all carry a copy of the ledger.
